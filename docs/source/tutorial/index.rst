#########################
Idris 2 速成课程
#########################

这是Idris 2的速成课程（有点像教程，但恐怕不那么温和！）。
它简要介绍了Idris语言的编程。
它涵盖了核心语言特性，假设读者有一些函数式编程语言（如Haskell或OCaml）的使用经验。

已经对Idris 1教程进行了修订和更新。自Idris 1以来更改的详细信息，请参阅:ref:`updates-index`。
